# Professional HRMS (Streamlit)

A complete HR Management System built using Python & Streamlit.

## Features
- Admin login
- Employee CRUD
- Attendance management
- Payroll automation
- HR analytics & charts

## Tech Stack
- Python
- Streamlit
- SQLite
- Pandas
- Matplotlib

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
